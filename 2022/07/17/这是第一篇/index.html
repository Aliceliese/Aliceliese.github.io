错误

博客执行hexo命令后出现HEXO: COMMAND NOT FOUND错误

将node_modules目录下hexo/bin添加到path环境变量中

在Node.js中

镜像

或许需要切换至原生镜像

npm get registry 查看镜像

npm config set registry http://registry.npm.taobao.org cnpm镜像

npm config set registry https://registry.npmjs.org/ 原

部署

在hexo目录下安装 hexo-deployer-git 插件

安装命令： npm install hexo-deployer-git –save

修改hexo的_config.yml

deploy:
type: git
repository: git@github.com:Aliceliese/Aliceliese.github.io.git
branch: master

hexo d 推送

主题

打开hexo目录下的_config.yml文件，在里面找到theme: landscape改为theme: blinkfox （blinkfox为我们要使用的主题名）,然后重新执行hexo g来重新生成

网站图标

把favicon拖入themes/你使用的主题/image文件夹中（不同主题的文件夹名可能会有不同）

打开themes/你使用的主题/_config.yml，找到favicon，修改成

-# favicon

favicon: /image/favicon.png

  • 想查看覆盖配置有没有生效，可以通过 hexo g --debug 查看命令行输出。
