A320冷舱启动

注意这些不是标准程序，只能让飞机发动机转起来

启动飞机前检查

模拟器里这些项目应该都处在检查完的状态

  1. Engine Master 1 and 2 Switch -> OFF
    • 主引擎 1, 2 都关闭
  2. Engine Mode Selector -> NORM
    • 引擎模式在 Normal 位置
  3. Weather Radar -> OFF
    • 关闭气象雷达
  4. Landing Gear -> Down
    • 起落架放下
  5. Wipers -> Both OFF
    • 两个雨刷都关闭

启动飞机

  1. Batt 1 and 2 -> ON
  2. EXT Power -> ON
    • 启动外部电源
    • 所有的仪表应该都已经亮起
  3. APU Fire Test
    • APU 火警测试
  4. APU Master and wait 3 sec -> ON
    • 打开 APU 主开关后等待 3 秒
  5. APU START -> ON
    • 启动APU
  6. Wait for APU START -> Green “AVAIL”
    • 等 APU 启动完成
  7. APU BLEED -> ON
    • 开启APU引气
  8. EXT Power -> OFF
    • 移除外部电源

上方仪表

由左至右，由下至上进行检查

左侧

  1. CREW SUPPLY -> ON
    • 打开乘员组氧气供给
  2. ADIRS 1, 2, 3 -> NAV
    • 开启三个惯性导航，需等待 6-7 分钟校准
    • 顺序为从左至右1,3,2

中间

  1. NAV & LOGO Light -> ON
    • 航行灯
  2. OVHD INTEG LT -> AS REQ.
    • 调整顶部仪表板背光 （应该在启动电池后调整）
  3. DOME -> DIM&BRT
    • 驾驶舱照明
  4. ANN LT -> TEST
    • 设置信号灯
  5. EMER EXIT LT -> ARM
    • 紧急出口灯
  6. SEAT BELTS -> ON
    • 安全带指示灯
  7. NO SMOKING -> AUTO
    • 禁止吸烟灯
  8. ALL PUMP -> ON
    • 燃油泵打开
  9. ENG 1 FIRE TEST
  10. ENG 2 FIRE TEST

右侧基本上不用动

MCDU未来单独写一篇说明了

后推和引擎启动前检查

等 IRS 校准完毕后后推(或者反推倒车滑出)

  1. 申请后推
  2. BECON Light -> ON
    • 启动 BECAON 灯
  3. Trust Levers -> IDLE
    • 确定油门在在 IDLE

启动引擎

  1. ENG MODE selector -> IGN/START
    • 把 ENG MODE selector 转至右侧引擎启动模式
  2. ENGINE 2 START -> ON
    • 先启动右侧的2号引擎
    • 等待启动完成后再启动1号引擎
    • 发动时应监控引擎数据，19% N1, 68% N2, 520°C EGT, 290 kg/h FF，等待 2 号引擎 AVAIL 出现
  3. ENGINE 1 START -> ON
    • 启动1号引擎，重复第 2 步直到 1 号引擎 AVAIL 出现

起飞准备

  1. ENG MODE selectgor -> NORM
    • 引擎模式切换回 NORM
  2. APU Bleed -> OFF
    • 关闭APU进气
  3. APU MASTER -> OFF
    • 关闭 APU
  4. FLAPS lever -> SET TAKEOFF POSITION
    • 放出襟翼到起飞需要位置
  5. NOSE Light -> Taxi
    • 开启滑行灯
  6. RWY TURN OFF -> ON
    • 开启转弯灯
  7. PARK BRK -> OFF
    • 关闭手刹
  8. AUTO BRK MAX -> ON
    • 开启最大自动刹车

启动完成，可以准备滑行起飞
